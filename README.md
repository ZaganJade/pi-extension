# Projects-PI

A collection of extensions for [pi](https://github.com/earendil-works/pi-mono), the coding agent.

## Extensions

| Package | Description | Install |
|---------|-------------|---------|
| [usage](./usage) | Claude Code-style `/usage` panel — 5H/weekly quota bars, cost/token breakdowns, live upstream provider quotas (ZAI, OpenAI Codex) | `pi install git:github.com/ZaganJade/pi-usage` |
| [multi-skill](./multi-skill) | Load multiple skills at once via `/skills` with autocomplete | `pi install git:github.com/ZaganJade/pi-multi-skill` |

## Install from local path

```bash
pi install ./usage
pi install ./multi-skill
```
